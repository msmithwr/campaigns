# Windows EC2 SDP Campaign 1 — One-Month Campaign Overview
## Windows Infrastructure Awareness & Cost Challenges (KSA/MENA Market)

## 1. Campaign Objectives

### Primary Goal
Position Cloudwrxs as the trusted Windows Server modernisation and Windows-to-AWS migration partner in KSA/MENA, using AWS Software Discount Program (SDP) savings as the opening value proposition.

### Campaign Window
**May 2026** — one-month campaign wave.

### Objectives
1. Generate high-value Windows Server migration and modernisation leads.
2. Promote the **Free Windows Estate Assessment** as the primary CTA.
3. Establish Cloudwrxs' authority in Windows infrastructure, licensing optimisation, HA/DR, and AWS migration.
4. Create qualified conversations for Windows Estate Assessment workshops.

## 2. Target Audiences & Core Messaging

### CFO / Finance Director
* Reduce Windows Server licensing and infrastructure costs by 35-60% through AWS SDP and right-sizing.
* Convert capex-heavy Windows refresh cycles into predictable opex.
* Improve financial visibility into Windows workload usage and cost drivers.

### CTO / Head of IT
* Modernise Windows Server environments for scale, performance, and resilience on AWS EC2.
* Use AWS SDP to reduce the licensing burden during cloud transformation.
* Strengthen Windows HA/DR and security posture with AWS-native services.

### IT Infrastructure Manager
* Simplify Windows operations with automation, monitoring, patching, backups, and DR.
* Reduce manual Windows maintenance and improve operational visibility.
* Improve stability for Windows workloads through AWS architecture patterns.
