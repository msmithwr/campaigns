# Windows EC2 SDP Campaign 2 — One-Month Campaign Overview
## Windows SDP Savings & Modernisation Benefits (KSA Market)

## 1. Campaign Objectives

### Primary Goal
Run the second Windows EC2 SDP wave in June 2026, building on Campaign 1 learnings and moving warm or newly segmented accounts toward TCO comparison and SDP Savings Calculator sessions.

### Campaign Window
**June 2026** — one-month campaign wave.

### Objectives
1. Generate high-value Windows migration leads for Windows Server 2019/2022 to AWS EC2.
2. Strengthen Cloudwrxs' authority in Windows SDP licensing optimisation and operational simplification.
3. Promote the **Windows-to-AWS TCO Comparison** and **SDP Savings Calculator Session** as primary CTAs.
4. Target IT Operations Managers and Procurement/Vendor Managers, with refreshed CTO and CFO messaging.
5. Convert Campaign 1 engagement into qualified Campaign 2 opportunities.

## 2. Target Audiences & Core Messaging

### IT Operations Manager
* Reduce Windows maintenance overhead through AWS automation.
* Improve patching, backup, monitoring, scaling, and disaster recovery for Windows workloads.
* Cut Windows licensing and infrastructure cost while improving operational resilience.

### Procurement / Vendor Manager
* Simplify Windows licensing and vendor management through AWS SDP.
* Improve compliance visibility and reduce audit risk.
* Convert unpredictable licensing costs into clearer consumption-based pricing.

### CTO / Head of IT
* Modernise Windows architecture for cloud readiness and digital transformation.
* Improve availability, performance, and governance for Windows workloads.
* Create a practical roadmap for Windows modernisation without overcommitting to a full migration project up front.

### CFO
* Reduce Windows total cost of ownership through SDP and right-sizing.
* Improve financial visibility and budget predictability.
* Reallocate Windows infrastructure savings to strategic digital initiatives.
